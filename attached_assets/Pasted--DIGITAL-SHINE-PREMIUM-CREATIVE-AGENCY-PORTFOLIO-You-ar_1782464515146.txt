# VERY IMPORTANT

First install and use **UI Pro Max**.

Execute:

```bash
npm install -g ui-ux-pro-max-cli
uipro init --ai codex
```

Use UI Pro Max components, layouts, animations and design patterns wherever appropriate.

Also install and use:

* Framer Motion
* Lenis (smooth scrolling)
* GSAP (for advanced text animations where needed)
* React Icons
* Tailwind CSS
* React Three Fiber (optional hero effects)
* Lucide Icons
